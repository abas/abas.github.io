Basic Tutorial Git + Github

2017-12-12
Cross-Platform
Cli, Pemrograman

Bismillah, kali ini kita akan belajar basic nge-Git :D

tapi sebelum itu kita pelajari dulu flow-nya atau alur program-nya ya sob :D jadi alur Git itu seperti ketika sobat download >< upload, bedanya dalam Git itu jadi push >< pull
alur penggunaan git seperti pada gambar dibawah

Inisialisasi -git init

berfungsi untuk menginisialisasi directory sebagai local repository, maksudnya kita akan membuat folder yang diinit sebagai local repository atau singkat nya folder yang akan di upload

Tracked File -git add {nama.file}/{nama.directory}

berfungsi untuk mengtrack new/modified file, jadi ketika ada file baru atau file yang termodifikasi maka untuk mengupload ke github repository kita perlu meng-track file tersebut baru bisa kita upload sob :D

Commit Message -git commit -m '{message}'

ketika kita push perubahan pada github tentu kita perlu menyertakan atau memberitahu apasih yang kita rubah disitu? jadi git commit itu berfungsi memberi pesan commit atau pesan perubahan yang terjadi pada github repository

Push Update -git push {nama.remote} {nama.branch}

untuk melakukan perubahan pada github repository kita gunakan command git push diatas terdapat origin yang berarti nama remote yang kita gunakan, remote disini yaitu alamat repository github yang akan kita update, sementara master adalah nama branch(cabang) yang kita gunakan
